👋 {{ '¿Qué es GTKY?' if lang == 'es' else 'What is GTKY?' }}

{% if lang == 'es' %} GTKY (Get To Know Ya, "conócete") es un juego para que un grupo se conozca. Cada quien responde preguntas sobre sí mismo, y después se hacen quizzes para ver qué tanto se conocen. Es la parte del rompehielos que normalmente no pasa — convertida en algo que sí te dan ganas de hacer. {% else %} GTKY (Get To Know Ya) is a get-to-know-you game for groups. Everyone answers a bunch of questions about themselves, then quizzes each other to see how well they actually know the people in the room. It's the part of the icebreaker that usually doesn't happen — turned into something you might actually want to do. {% endif %}

👥 {{ '¿Para quién es?' if lang == 'es' else "Who's it for?" }}

{% if lang == 'es' %} Grupos de 5 a 50 personas que van a pasar tiempo juntas — retiros de equipo, salones de clase, reuniones familiares, onboarding. Comparte la URL con todos los que estén en la misma WiFi y cada quien puede entrar desde su propio celular. {% else %} Groups of 5 to 50 people who'll be spending time together — team offsites, classrooms, family reunions, onboarding. Share the URL with everyone on the same WiFi and each person can join from their own phone. {% endif %}

▶️ {{ 'Cómo funciona' if lang == 'es' else 'How it works' }}

  1. {% if lang == 'es' %} Entra con tu nombre. Si ya usaste GTKY antes, toca "Ya estoy aquí". {% else %} Sign in with your name. If you've used GTKY before, tap "I'm already here" instead. {% endif %}
  2. {% if lang == 'es' %} Responde preguntas sobre ti — preferencias, opiniones, qué prefieres. {% else %} Answer survey questions about yourself — preferences, opinions, would-you-rathers. {% endif %}
  3. {% if lang == 'es' %} Cuando hayas respondido unas ocho, puedes empezar los quizzes. {% else %} Once you've answered eight or so, you can take quizzes. {% endif %}
  4. {% if lang == 'es' %} Revisa Conexiones para ver qué tanto conoces a cada quien. {% else %} Check Connections to see how well you know each person. {% endif %}
  5. {% if lang == 'es' %} Toca cualquier cara para ver qué tanto conoces a esa persona y qué tanto te conoce. {% else %} Tap any face to see how well you know that person and how well they know you. {% endif %}

💡 {{ 'Tips y preguntas' if lang == 'es' else 'Tips & FAQ' }}

{{ '¿Quién ve mis respuestas?' if lang == 'es' else 'Who sees my answers?' }}

{% if lang == 'es' %} Nadie ve tu lista completa de respuestas — ni siquiera tú, después de responderlas. Los demás solo se enteran de tus respuestas una por una, tomando quizzes sobre ti. Solo el admin puede ver la lista completa de respuestas de alguien. {% else %} No one sees your full list of answers — not even you, after you submit them. Other players only learn your answers one at a time, by taking quizzes about you. Only the admin can see anyone's full answer list. {% endif %}

{{ '¿Quién ve mi foto?' if lang == 'es' else 'Who sees my photo?' }}

{% if lang == 'es' %} Solo las otras personas en este mismo servidor. Las fotos se guardan localmente y no se suben a internet. {% else %} Only other people on this same server. Photos are stored locally and never uploaded anywhere else. {% endif %}

{{ '¿Puedo cambiar mis respuestas?' if lang == 'es' else 'Can I change my answers?' }}

{% if lang == 'es' %} Por ahora, las respuestas son permanentes. Pide al admin que reinicie tu cuenta si necesitas empezar de nuevo. {% else %} Currently answers are permanent. Ask an admin to reset your account if you need to start over. {% endif %}

{{ '¿Cómo funciona la puntuación del quiz?' if lang == 'es' else 'How does quiz scoring work?' }}

{% if lang == 'es' %} Cada respuesta correcta vale un punto. Las puntuaciones en Conexiones son el porcentaje de aciertos. {% else %} Each correct answer is worth one point. Connection scores are your percentage correct. {% endif %}

{{ '¿Quién es el admin?' if lang == 'es' else 'Who is the admin?' }}

{% if lang == 'es' %} Quien sepa el PIN. El PIN por defecto es 1234 — cámbialo en cuanto puedas. {% else %} Whoever knows the PIN. The default PIN is 1234 — change it as soon as you can. {% endif %}
